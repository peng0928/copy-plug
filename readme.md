# 🦄 Copy Bypass Hook

[![Chrome Web Store](https://img.shields.io/badge/Chrome-Extension-blue.svg?logo=googlechrome)](https://chrome.google.com/webstore)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Version](https://img.shields.io/badge/Version-1.0.0-orange.svg)]()

一个可爱的 Chrome 浏览器扩展，通过 Hook 技术解除网页的复制限制，让你自由复制任何内容。

![Preview](images/image.png)

## ✨ 功能特性

- 🎯 **一键解除复制限制** - 绕过 JavaScript 禁止复制、右键菜单限制
- 🎨 **可爱界面设计** - 萌系图标 + 渐变悬浮通知
- ⌨️ **快捷键操作** - `Ctrl+E` 快速开启/关闭
- 🔄 **状态记忆** - 刷新页面自动恢复状态
- 🛡️ **隐私安全** - 纯本地运行，不收集任何数据
